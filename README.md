# Project
Мы представляем проект по разработке 2D-игры "Ягодное Приключение". Основная цель проекта - создание полноценной браузерной игры с системой уровней и прогрессирующей сложностью.

Ключевые особенности игры:

· Механика собирания ягод для прогресии
· Многоуровневая структура с увеличивающейся сложностью
· Динамическое движение врагов и персонажа
· Система счета и достижения целей уровня

Техническая реализация:

· Кастомная система физики
· Поэтапное увеличение количества ягод для сбора
· Адаптивная сложность врагов между уровнями
· Интуитивное управление и визуальная обратная связь
